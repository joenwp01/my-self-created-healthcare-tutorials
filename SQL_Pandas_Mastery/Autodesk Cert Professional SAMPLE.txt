Autodesk Certified Professional in Civil 3D for Infrastructure Design:

Autodesk Certified Professional in Civil 3D for Infrastructure Design. Autodesk’s current objective guide breaks it into six major content areas: Alignments, Profiles, and Sections (21%); Surfaces and Grading (20%); Data Management and Collaboration (19%); Corridors (16%); Pipe and Pressure Networks (12%); and Points, Parcels, and Survey Data (12%).

At a practical level, that means you should expect questions around:

Alignments / profiles / sections: creating, editing, and labeling alignments, profiles, profile views, sample lines, and section views/sheets.
Surfaces / grading: TIN surfaces, surface analysis, extracting surface data, grading tools, and feature line elevation control.
Pipe / pressure networks: part catalogs and parts lists, creating networks, modifying pressure pipes, adding parts in plan/profile/section, and labeling networks.
Data management / collaboration: Data Shortcuts, Data References vs Xrefs, reference templates, plan production sheets, reports, import/export formats, sites, styles, coordinate systems, and shared reference points with Revit.
Points / parcels / survey: COGO points, point styles and labels, point groups, parcel creation/layout/labels, and survey-related data handling.
Corridors: corridor regions, targets, frequency, assemblies, subassemblies, and corridor editing.


A few examples of the kind of “calculation” you should be ready for are:

- figuring which grade or slope is correct from profile information,
- recognizing the correct station/offset or label type,
- interpreting cut vs. fill or which surface should be base versus comparison,
- checking whether a pipe slope, depth, invert, or diameter setup makes sense,
- reasoning through parcel area/frontage or equal-segment subdivision logic.
NOTE: So I would prepare for practical civil-design math, not heavy hand computation. Think short numeric reasoning and reading design conditions, rather than long formulas or full survey/structural calculations.

SAMPLE MULIPLE CHOICE QUESTIONS:
 
Below are original Civil 3D-style practice problems modeled on Autodesk’s published domains for the current Civil 3D professional exam: alignments/profiles/sections, surfaces/grading, pipe and pressure networks, data management/collaboration, points/parcels/survey, and corridors.

Sample problems

1) Multiple choice — Alignments
You created an alignment from a polyline and notice the stationing runs opposite the intended project direction. What is the best fix?

A. Rebuild the surface
B. Reverse the alignment direction
C. Create an offset alignment
D. Swap the label set

Answer: B

2) Multiple choice — Profiles
A design profile must transition from a +2.0% grade to a -1.5% grade while maintaining smooth vertical geometry. Which Civil 3D concept is most directly involved?

A. Sample line group
B. Vertical curve and PVI editing
C. Data shortcut reference
D. Section view band set

Answer: B

3) Multiple choice — Station/offset labels
Which label type is most appropriate when you need a label tied to a specific fixed location rather than dynamically following a changing offset relationship?

A. General line label
B. Station offset fixed point label
C. Station equation label
D. Curve table tag

Answer: B

4) Multiple choice — Surfaces
You need to compare existing ground to proposed grade and calculate cut/fill. Which surface object is typically used for that comparison?

A. Grid surface
B. TIN volume surface
C. Corridor surface only
D. Watershed analysis surface

Answer: B

5) Multiple choice — Feature lines / grading
Two feature lines are placed in the same site and begin affecting one another unexpectedly. What is the most likely reason?

A. Their label styles conflict
B. Objects in the same site can interact geometrically
C. The drawing has the wrong coordinate system
D. The point groups are out of date

Answer: B

6) Multiple choice — Pipe networks
A gravity pipe network needs to reference the ground surface and enforce design behavior for cover/depth. Which configuration area is most relevant?

A. Parcel properties
B. Pressure part catalog path
C. Network rules / reference surface settings
D. Surface analysis tab

Answer: C

7) Multiple choice — Data management
When should you prefer a Data Reference over a simple Xref?

A. When you only need linework appearance
B. When you need Civil 3D object relationships maintained across files
C. When plotting to PDF only
D. When using plain AutoCAD blocks

Answer: B

8) Multiple choice — Plan production
What primarily controls the sheet layout, viewport arrangement, and displayed content when creating sheets from view frames?

A. Point group order
B. The sheet/template setup used by the plan production tools
C. The survey database
D. The pressure network catalog

Answer: B

9) Multiple choice — Points / survey
Which statement best distinguishes COGO points from survey points?

A. COGO points cannot be labeled
B. Survey points are just blocks with attributes
C. Survey points are tied to survey-database workflows, while COGO points are standard Civil 3D points created/imported directly in the drawing
D. COGO points only work in profile view

Answer: C

10) Multiple choice — Parcels
You need to divide a parcel into several equal-width lots but keep any leftover width distributed according to layout logic. Which parcel workflow is most relevant?

A. Parcel renumbering
B. Parcel subdivision into equal segments with remainder handling
C. Surface boundary extraction
D. Corridor target mapping

Answer: B

11) Multiple choice — Corridors
You need one assembly for a typical roadway section and a different assembly near an intersection, all along the same baseline. What is the usual Civil 3D approach?

A. Create two separate drawings only
B. Split the corridor into regions and assign different assemblies
C. Convert the corridor to a surface
D. Replace all subassemblies with feature lines

NOTE:
These practice items are closest to the official objective areas Autodesk lists:

1 - Alignments, Profiles, and Sections
    Includes alignment/profile/section creation and labeling, profile-grade or slope reasoning, and station/offset label recognition.

2 - Surfaces and Grading
    Includes feature lines, slope/spot/contour interpretation, cut vs. fill, base vs. comparison surface selection, surface analysis, and volume comparison.

3 - Pipe and Pressure Networks
    Includes pipe slope, depth, invert, diameter, flow direction, parts lists/catalogs, and plan/profile/section network behavior.

4 - Data Management and Collaboration
    Includes data shortcuts, data references, templates, plan production, sheet-related workflows, coordinate systems, and collaboration setup.

5 - Points, Parcels, and Survey Data
    Includes parcel area/frontage reasoning, equal-segment subdivision logic, points, point groups, and survey-data workflows.

6 - Corridors
    Includes corridor regions, targets, frequency, assemblies, and related corridor behavior.
