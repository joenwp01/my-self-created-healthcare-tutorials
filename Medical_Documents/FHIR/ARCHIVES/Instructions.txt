Best way to place it on your system:

Put the extracted folder in a permanent location that you will not rename or move often. Good examples are:

C:\Users\YourName\Documents\FHIR-Demo
C:\Users\YourName\Desktop\FHIR-Demo

Best way to open and bookmark it:

Method 1 is the easiest. Extract the ZIP, open index.html, and then bookmark that page in Chrome, Edge, or Firefox. That bookmark will point to a local file path. This works well as long as the folder stays in the same place.

Method 2 is the cleanest. Extract the ZIP, then run the small local server from that folder:

On Windows, double-click run_local_server.bat
Then open http://localhost:8000
Bookmark that page

I recommend Method 2 if you expect to use it repeatedly, because the bookmark looks cleaner and behaves more like a normal website. I recommend Method 1 if you want the fewest steps and no terminal/server behavior.

Practical recommendation for you:
